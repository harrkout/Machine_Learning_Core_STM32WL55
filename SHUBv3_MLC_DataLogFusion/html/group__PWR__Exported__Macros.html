Clear a specific PWR flag.

Note
Clearing of flags {PWR_FLAG_STOP, PWR_FLAG_STOP2, PWR_FLAG_SB} are grouped: clearing of one flag also clears the other ones.
Parameters
__FLAG__specifies the flag to clear. This parameter can be one of the following values:

/-----------------------------—SCR (SRR)---------------------------—/

  • PWR_FLAG_WU Wake Up Flag of all pins.
  • PWR_FLAG_WUF1 Wake Up Flag 1. Indicates that a wakeup event was received from the WKUP pin 1.
  • PWR_FLAG_WUF2 Wake Up Flag 2. Indicates that a wakeup event was received from the WKUP pin 2.
  • PWR_FLAG_WUF3 Wake Up Flag 3. Indicates that a wakeup event was received from the WKUP pin 3.
  • PWR_FLAG_WPVD Wakeup PVD flag
  • PWR_FLAG_WRFBUSY Wake-up radio busy flag (triggered status: wake-up event or interruption occurred at least once. Can be cleared by software)

/-------------------------—EXTSCR-----------------------—/

  • PWR_FLAG_LPMODES System Standby Flag for CPU1.
Return values
None

Check whether or not a specific PWR flag is set.

Parameters
__FLAG__specifies the flag to check. This parameter can be one of the following values:

/-----------------------------—SR1----------------------------—/

  • PWR_FLAG_WUF1 Wake Up Flag 1. Indicates that a wakeup event was received from the WKUP pin 1.
  • PWR_FLAG_WUF2 Wake Up Flag 2. Indicates that a wakeup event was received from the WKUP pin 2.
  • PWR_FLAG_WUF3 Wake Up Flag 3. Indicates that a wakeup event was received from the WKUP pin 3.
  • PWR_FLAG_WPVD Wakeup PVD flag
  • PWR_FLAG_WUFI Wake-Up Flag Internal. Set when a wakeup is detected on the internal wakeup line.
  • PWR_FLAG_WRFBUSY Wake-up radio busy flag (triggered status: wake-up event or interruption occurred at least once. Can be cleared by software)

/-----------------------------—SR2----------------------------—/

  • PWR_FLAG_LDORDY Main LDO ready flag
  • PWR_FLAG_SMPSRDY SMPS ready Flag
  • PWR_FLAG_REGLPS Low-power Regulator 1 started: Indicates whether the regulator is ready after a power-on reset or a Standby/Shutdown.
  • PWR_FLAG_REGLPF Low-power Regulator 1 flag: Indicates whether the regulator 1 is in main mode or is in low-power mode.
  • PWR_FLAG_REGMRS Low-power regulator (main regulator or low-power regulator used) flag.
  • PWR_FLAG_FLASHRDY Flash ready flag
  • PWR_FLAG_VOSF Voltage Scaling Flag. Indicates whether the regulator is ready in the selected voltage range or is still changing to the required voltage level.
  • PWR_FLAG_PVDO Power Voltage Detector Output. Indicates whether VDD voltage is below or above the selected PVD threshold.
  • PWR_FLAG_PVMO3 Peripheral Voltage Monitoring Output 3. Indicates whether VDDA voltage is is below or above PVM3 threshold.
  • PWR_FLAG_RFEOL Indicate whether supply voltage is below radio operating level (radio "end of life").
  • PWR_FLAG_RFBUSYS Radio busy signal flag (current status).
  • PWR_FLAG_RFBUSYMS Radio busy masked signal flag (current status).

/-------------------------—EXTSCR-----------------------—/

  • PWR_FLAG_STOP System Stop 0 or Stop1 Flag for CPU1.
  • PWR_FLAG_STOP2 System Stop 2 Flag for CPU1.
  • PWR_FLAG_SB System Standby Flag for CPU1.
  • PWR_FLAG_C1DEEPSLEEP CPU1 DeepSleep Flag.
Return values
Thenew state of FLAG (TRUE or FALSE).
