Detailed Description

Elements values convention: 0000 0000 0XXY YYYYb

  • Y YYYY : Flag position in the XX register (5 bits)
  • XX : Status register (2 bits)
    • 01: SR1 register
    • 10: SR2 register
    • 11: EXTSCR register
