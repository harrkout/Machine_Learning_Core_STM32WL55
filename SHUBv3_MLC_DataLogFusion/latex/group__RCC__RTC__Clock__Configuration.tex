Macros to enable or disable the RTC clock. 

\begin{DoxyNote}{Note}
As the RTC is in the Backup domain and write access is denied to this domain after reset, you have to enable write access using \doxylink{group__PWR__Exported__Functions__Group1_ga3d07cef39bf294db4aed7e06e5dbf9af}{HAL\+\_\+\+PWR\+\_\+\+Enable\+Bk\+Up\+Access()} function before to configure the RTC (to be done once after reset). 

These macros must be used after the RTC clock source was selected. 
\end{DoxyNote}
