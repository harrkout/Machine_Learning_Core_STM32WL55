◆ __HAL_RTC_TAMPER_CLEAR_FLAG

#define __HAL_RTC_TAMPER_CLEAR_FLAG ( __HANDLE__,
__FLAG__ )   (((TAMP->SCR) = (__FLAG__)))

Clear the RTC Tamper's pending flags.

Parameters
__HANDLE__specifies the RTC handle.
__FLAG__specifies the RTC Tamper Flag to clear. This parameter can be:
  • RTC_FLAG_TAMP_ALL: All tampers flag
  • RTC_FLAG_TAMP_1: Tamper1 flag
  • RTC_FLAG_TAMP_2: Tamper2 flag
  • RTC_FLAG_TAMP_3: Tamper3 flag
  • RTC_FLAG_INT_TAMP_ALL: All Internal Tamper flags
  • RTC_FLAG_INT_TAMP_3: Internal Tamper3 interrupt
  • RTC_FLAG_INT_TAMP_5: Internal Tamper5 interrupt
  • RTC_FLAG_INT_TAMP_6: Internal Tamper6 interrupt
  • RTC_FLAG_INT_TAMP_8: Internal Tamper8 interrupt
Return values
None
